# ReactChat-App


## Project Description

This project is centered around front-end development using React.js, with CSS utilized for styling. The application leverages both React Context API and Redux for robust state management.

The chat app includes essential features such as user authentication (login, register, reset password), messaging (chat, conversation, group calls), contact management, starred messages, media handling, file and link sharing, user settings, profile editing, and emoji pickers. Users can also toggle between dark and light modes and customize theme colors through the settings icon located in the bottom-left corner.

Future enhancements will focus on integrating a backend to expand the app's functionality.

## Requirements

- Basic ReactJs knowledge
- Basic HTML, CSS knowledge

## Getting Started

After getting the project files, you need to go the file directory and run

```shell
npm install
```

and after that start the dev server.

```shell
npm run start
```
